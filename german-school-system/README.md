# German School System Guide – Essen

A plain-language reference for families, newcomers, and anyone wanting to understand
how German education works — from first school days to university graduation, with a
special focus on **Essen** and the surrounding Ruhr metropolitan area.

## Contents

| File | Description |
|------|-------------|
| `Essen_German_Schools_Guide.pdf` | Ready-to-read 15-page PDF guide |
| `essen_schools_guide.py` | Python script that generates the PDF (uses ReportLab) |

## What the PDF Covers

| Chapter | Topic |
|---------|-------|
| 1 | German School System – Overview & pathway table |
| 2 | School types explained simply (Grundschule, Gymnasium, Realschule, Hauptschule, Gesamtschule) |
| 3 | From Abitur to University (IB, NC grades, tuition, degree lengths) |
| 4 | Vocational Training & the Dual System (Ausbildung, Meister) |
| 5 | 11 Universities & Colleges near Essen with distances & specialisations |
| 6 | Top 10 Gymnasiums near Heinickestraße, Essen — profiles & reviews |
| 7 | Gymnasium comparison table (bilingual, IB, sports, arts, MINT ratings) |
| 8 | Glossary of 24 key German education terms |

## Reference Point

All school distances are measured from **Heinickestraße 8, 45128 Essen** (Innenstadt-Süd).

## Regenerating the PDF

```bash
pip install reportlab
python german-school-system/essen_schools_guide.py
```

The PDF is written to `german-school-system/Essen_German_Schools_Guide.pdf`.
